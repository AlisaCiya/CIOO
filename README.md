## Dataset Format

All datasets in this project follow the **YOLO format**.

Each dataset should be organized as follows:

```text
<dataset_name>/
├── images/
│   ├── train/
│   └── val/
└── labels/
    ├── train/
    └── val/
```

Replace `<dataset_name>` with the actual dataset name (e.g., `VOC`, `COCO`, `VisDrone`).
