### VOC

```text
VOC/
├── images/
│   ├── train/

    ├── train/
    └── val/
```

| Directory | Description |
|-----------|-------------|

| `images/val/` | Validation images |
| `labels/train/` | Training annotations |
| `labels/val/` | Validation annotations |
